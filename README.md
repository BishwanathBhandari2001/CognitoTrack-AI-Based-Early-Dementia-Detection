# CognitoTrack: AI-Based Early Dementia Detection Using Smart Device Interaction Patterns

## Overview

CognitoTrack is an AI-powered early dementia detection system that analyzes speech and smart device interaction patterns to identify early signs of cognitive decline.

The system uses speech analysis, behavioral monitoring, and deep learning techniques to detect subtle changes associated with dementia.

This research was published in the International Journal of Creative Research Thoughts (IJCRT).

## Problem Statement

Dementia often goes undetected during its early stages.

Traditional diagnostic methods are expensive, time-consuming, and require specialized healthcare infrastructure.

This project aims to provide a scalable, non-invasive, AI-powered solution for early dementia screening through speech analysis and smart device interactions.

## Methodology

### Data Preprocessing

- Noise Reduction
- Audio Segmentation
- Feature Normalization

### Feature Extraction

- Mel Frequency Cepstral Coefficients (MFCC)
- Log-Mel Spectrograms

### Deep Learning Model

Bidirectional Long Short-Term Memory (BiLSTM)

The model captures temporal dependencies in speech signals and identifies cognitive decline patterns.

## Results

### Model Accuracy

- Test Accuracy: 78%
- Control Class Accuracy: 84.38%
- Dementia Class Accuracy: 80%

### Performance Metrics

- Accuracy
- Precision
- Recall
- F1 Score

The results demonstrate the effectiveness of speech-based AI models for early dementia detection.
